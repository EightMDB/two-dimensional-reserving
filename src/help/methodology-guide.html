Methodology Guide

Comprehensive documentation for all reserving methodologies

When to Use Each Method

Chain Ladder

Best For:
  • Stable loss development patterns
  • Sufficient historical data
  • Mature lines of business
Considerations:
  • Assumes past patterns continue
  • Sensitive to outliers in recent periods
  • May be unstable for immature data
View Method →

Bornhuetter-Ferguson

Best For:
  • New or emerging lines
  • When a priori loss ratios are available
  • Unstable development patterns
Considerations:
  • Requires credible expected loss ratio
  • Less responsive to recent trends
  • Balances between data and expectations
View Method →

Benktander

Best For:
  • Iterative convergence between CL and BF
  • When both methods are appropriate
  • Seeking middle-ground estimate
Considerations:
  • Typically uses 1-2 iterations
  • Combines strengths of CL and BF
  • Requires same inputs as BF
View Method →

Cape Cod

Best For:
  • Estimating implied loss ratio
  • Varying exposure by period
  • Limited a priori information
Considerations:
  • Assumes constant loss ratio across periods
  • Requires exposure base data
  • Data-driven loss ratio estimate
View Method →
